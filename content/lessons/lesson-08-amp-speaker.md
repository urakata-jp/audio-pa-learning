# アンプとスピーカーの選び方

## このレッスンで学ぶこと

- パッシブスピーカーとパワードスピーカーの違い
- アンプ出力、スピーカー許容入力、インピーダンス
- スピーカー感度と必要音量
- 距離、人数、屋内外から考える選定の目安
- W数だけで判断しない理由

## 導入会話

:::dialogue
初心者:  
「100人の屋外イベントなら、何Wのスピーカーを選べばいいですか？」

講師:  
「W数だけでは決まりません。必要な音量、距離、屋内外、スピーカー感度、音楽中心かスピーチ中心かを合わせて考えます。」
:::

## 基本解説

スピーカーには大きく2種類あります。

| 種類 | 特徴 |
|---|---|
| パワードスピーカー | アンプ内蔵。接続がシンプル |
| パッシブスピーカー | 外部アンプが必要。アンプとスピーカーの組み合わせを考える必要がある |

パッシブスピーカーでは、アンプ出力、スピーカー許容入力、インピーダンスを確認します。

## 主なスペック

| スペック | 意味 |
|---|---|
| W数 | どれくらいの電力を扱えるか、またはアンプが出せるか |
| インピーダンス | 電気的な負荷。一般的には4Ω、8Ωなど |
| 感度 | 1W/1mでどれくらいの音圧が出るか |
| 最大SPL | スピーカーが出せる最大音圧の目安 |
| 指向角 | 音が広がる範囲 |

:::point
同じW数でも、感度が高いスピーカーの方が大きな音を出しやすい場合があります。W数だけで選ばないことが重要です。
:::

## 必要アンプ出力の考え方

必要な出力は、以下で変わります。

- 目標SPL
- 一番遠い聴取位置までの距離
- スピーカー感度
- 必要なヘッドルーム
- 屋内か屋外か
- スピーチ中心か、音楽中心か

Crownの計算ツールでは、聴取距離、目標SPL、スピーカー感度、ヘッドルームから必要なアンプ出力を概算できます。

## ざっくりした現場目安

以下はあくまで初期検討の目安です。

| イベント | 考え方 |
|---|---|
| 講演会 | 明瞭度重視。過度な低音や大音量は不要 |
| 歌・出し物 | スピーチより余裕が必要。音楽再生や歌のピークに注意 |
| 屋外 | 反射が少なく音が逃げる。距離減衰を考慮 |
| 体育館・講堂 | 残響が多い場合は音量より明瞭度を優先 |

## よくある失敗

- W数だけで選ぶ
- スピーカー感度を見ない
- 屋外なのに室内と同じ感覚で選ぶ
- 小型スピーカーに低音や音楽の迫力を求めすぎる
- アンプとスピーカーのインピーダンスを確認しない
- 前方だけ大音量で後方が聞こえない

## 現場チェックリスト

- [ ] 屋内か屋外か確認した
- [ ] 客席の最大距離を確認した
- [ ] スピーチ中心か音楽中心か確認した
- [ ] スピーカー感度を確認した
- [ ] 最大SPLを確認した
- [ ] アンプとスピーカーのインピーダンスを確認した
- [ ] 必要なら補助スピーカーを検討した

## まとめ

アンプとスピーカーは、W数だけでなく感度、最大SPL、距離、用途、屋内外、ヘッドルームを含めて考えます。小中規模現場でも、機材選定を間違えるとハウリング、音割れ、聞こえにくさにつながります。
