## 導入会話

:::dialogue
初心者:  
「音が小さい時は、ゲインを上げればいいんですよね？」

講師:  
「ゲインは入口の調整なので、むやみに上げるとノイズやハウリング、音割れにつながります。まずは信号の流れとメーターを見ながら調整します。」
:::

## 基本解説

ミキサーは複数の音をまとめ、音量や音質、送り先を調整する機材です。

主な操作部は以下です。

| 項目 | 役割 |
|---|---|
| ゲイン | 入力信号の入口の感度を決める |
| フェーダー | そのチャンネルをどれくらい出すか決める |
| EQ | 音質を調整する。基本は不要な帯域を削る |
| ミュート | 音を一時的に止める |
| メーター | 信号の大きさを確認する |
| PFL/SOLO | 本番出力に出さずに確認する機能 |

[diagram: mixer-channel-strip]

## ゲインとフェーダーの違い

ゲインは入口、フェーダーはミックス上のバランスです。

```text
マイク → ゲイン → EQ → フェーダー → MAIN/AUX
```

ゲインが低すぎると、フェーダーを上げても音が小さく、ノイズが目立ちやすくなります。ゲインが高すぎると、音が割れたりハウリングしやすくなります。

フェーダーは、各チャンネルのバランス調整です。講演者の声、司会者の声、PC音源、BGMなどの相対的な音量を決めます。

:::point
ゲインは本番中に頻繁に触る場所ではありません。リハーサル時に適正に合わせ、本番中は基本的にフェーダーで調整します。
:::

## ヘッドルーム

ヘッドルームとは、急に大きな声や音が入った時に音割れしないための余裕です。普段の声でメーターが適正でも、笑い声、大声、歌、拍手で一気に大きくなることがあります。

## 現場での使い方

音が小さい時は、以下の順番で見ます。

1. マイクが口から離れていないか
2. 入力メーターが振れているか
3. ゲインが低すぎないか
4. チャンネルミュートが解除されているか
5. フェーダーが上がっているか
6. MAIN/AUXへの送りがあるか
7. スピーカーやアンプ側の音量が適切か
8. ハウリングしない位置関係か

## よくある失敗

- 音が小さい原因を確認せずゲインを上げる
- PC音源だけ極端に大きい
- AUXへ送っているつもりが送れていない
- ミュートやDCAで音が止まっていることに気づかない
- メーターを見ずに耳だけで判断する

## 現場チェックリスト

- [ ] 各チャンネルに名前を付けた
- [ ] 入力メーターを確認した
- [ ] ゲインをリハーサルで合わせた
- [ ] フェーダー位置を基準値に戻した
- [ ] ミュート状態を確認した
- [ ] MAINとAUXへの送りを確認した

## まとめ

ミキサー操作で一番大事なのは、ゲインとフェーダーを分けて考えることです。ゲインは入口の適正化、フェーダーは本番中のバランス調整です。音が小さいからといって、すぐにゲインを上げるのではなく、信号の流れとメーターを見ながら判断します。
